This plugin helps you to administrate application of Sina microblog

== Description ==

访客使用新浪微博账号登录你的博客发表评论，
使用新浪微博的头像，
同步留言到新浪微博。
新浪微博登录标志和评论同步提示标签可自定义位置。
访客自动获得本地同名新账户
访客本地账户前缀可自定义，默认sina_，防止重名或者伪造用户
访客登录获得本地账户后得到新浪微博私信通知
访客自动在新浪微博关注你的新浪微博。
新文章发布时可以在新浪微博发布消息。
本插件曾借鉴denishua的插件。

== Installation ==

安装后启用，进入后台“设置”下的“管理新浪微博”，设定网站管理员在新浪微博的账户，作为你的官方新浪微博
其他参数默认即可良好工作，具体设置参考页面说明文字
